### Persiapan
**Update dulu**
```bash
sudo apt update && sudo apt upgrade -y
```

**Intall dependencies**
```bash
sudo apt-get install jq unzip -y
```

**Screen installation (jika dibutuhkan**
```bash
sudo apt install screen
```

**Instal Git**
```bash
apt install git
```

**Instal Cargo**
```bash
apt install cargo
```
